# Joby Aviation Menu Animation

A sleek, premium menu navigation animation inspired by the **Awwwards Site of the Day** (awarded Mar 16, 2026) by **TinyWins**, originally based on [Joby Aviation](https://www.jobyaviation.com/). Now branded as **Unearth**, this project was built with modern web technologies including Vite, GSAP, and Split-Type.

![Website Preview](/public/assets/website%20Preview%20Img.png)

## 🚀 Features

- **Smooth GSAP Timelines**: Advanced sequencing for menu transitions.
- **Multilayered Backgrounds**: Staggered panel animations for a dynamic entrance.
- **Typography Splitting**: Smooth text revealing effects using `split-type`.
- **Responsive Design**: Optimized for different screen sizes.
- **Vite Powered**: Extremely fast development and optimized production builds.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Custom styling with CSS variables.
- **GSAP (GreenSock Animation Platform)**: Industry-standard animation library.
- **Split-Type**: Text manipulation for reveal animations.
- **Vite**: Modern frontend tooling.

## ⚙️ Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run in development mode**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
